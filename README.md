## ğŸ–¥ï¸ Running the Server

Follow these specific commands to compile contracts and start the local blockchain network.

### 1. Compile Smart Contracts
Force compile the contracts to generate the necessary artifacts.
```bash
npx hardhat compile --force
```
### 2. Start the Local Mode
Navigate to the voting app directory and start the Hardhat node.
```bash
cd voting-app
npx hardhat node
```
Note: Keep this terminal window open to keep the local network running.

### 3.Deploy Contracts
Open a new terminal, return to the project root, and deploy the scripts to localhost.
```bash
npx hardhat run scripts\deploy.js --network localhost
```
